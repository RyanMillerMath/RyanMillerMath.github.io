Single-Variable Calculus

This web app utilizes the Newton API to perform single-variable calculus and to simplify the results.

The code repo for this project is here.

Languages/Frameworks Used

Sass/SCSS, ES6/ES2015, React/JSX, Axios

Knowledge This App Shows

Besides my deep love for mathematics, my calculus app shows core React development principles: reusable components, having one source of statefulness, and passing down statefulness through props. As a React app, this app naturally expresses JSX and various features of ES6, including classes, importing, and default exports. Additionally, this app utilizes various aspects of Sass for styling, including variables, partials, and mixins. This app also uses Axios for promise-based AJAX requests.

Phonebook

This fully functional phonebook allows you to create contacts, view contacts' information, and update and delete existing contacts.

The code repo for this project is here.

Languages/Frameworks Used

HTML5, CSS3, ES6/ES2015, Angular

Knowledge This App Shows

My CRUD phonebook app presents the most integral concepts of Angular: modules, components (including reusable components), services and dependency injection, and pipes. Additionally, this app exemplifies several useful features of Angular: data binding (both one way and two way), routing, and Angular's http module. Naturally then, this app also expresses non-promise based AJAX requests and reactive programming with RxJS through observables. This app also shows ES6's new template string feature in the super useful case of interpolating URL segments.
